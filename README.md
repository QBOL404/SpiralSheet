### Python App
.wav file to continuous spiral

module you need:
numpy
tkinter
pyinstaller
matplotlib
librosa
os
pygame

to make .exe: 
>> pyinstaller -w cont_spiral.py

to use:
.mp3 .wav 있는 폴더 선택
음원 선택 후 execute
